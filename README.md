# URL Shortener

This was a guided project from DIO's instructor [Alexia Dorneles](https://github.com/alexiadorneles).
<br><br>
### Available Scripts:

```sh
npm run build:watch
```

```sh
npm run dev
```

Open http://localhost:5000 to view it in your browser.
<br><br>
### Dependencies:
- @typegoose/typegoose
- cors
- express
- mongoose
- shortid
